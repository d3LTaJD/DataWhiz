üìà Statistical Analysis Features

Descriptive Statistics

Get instant insights with comprehensive descriptive statistics:

  • Central Tendency: Mean, median, mode calculations
  • Dispersion Measures: Variance, standard deviation, range
  • Percentiles: Quartiles, deciles, and custom percentiles
  • Distribution Summary: Skewness and kurtosis analysis

Distribution Analysis

Understand your data distribution patterns:

  • Normality Tests: Shapiro-Wilk, Kolmogorov-Smirnov tests
  • Distribution Identification: Identify Gaussian, exponential, log-normal patterns
  • Outlier Detection: IQR method, Z-score analysis
  • Histogram Analysis: Visual distribution analysis

Correlation Analysis

Discover relationships between variables:

  • Pearson Correlation: Linear relationships for continuous data
  • Spearman Correlation: Rank-based relationships for ordinal data
  • Correlation Matrix: Visualize all variable relationships
  • Significance Testing: P-values and confidence intervals

Regression Analysis

Model and predict relationships:

  • Linear Regression: Simple and multiple linear models
  • Logistic Regression: Binary outcome predictions
  • Model Evaluation: R¬≤, adjusted R¬≤, RMSE metrics
  • Residual Analysis: Diagnostic plots and validation

Time Series Analysis

Analyze temporal patterns and trends:

  • Trend Analysis: Identify upward, downward, or stationary trends
  • Seasonality Detection: Seasonal patterns and cycles
  • Forecasting: Future value predictions
  • Volatility Analysis: Variance and risk assessment
