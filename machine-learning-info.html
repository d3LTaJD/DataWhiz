üß† Available ML Models

Random Forest (Classifier & Regressor)

Type: Ensemble Learning Algorithm

Best For: High-accuracy predictions, handling complex patterns

  • Creates multiple decision trees and combines their predictions
  • Resistant to overfitting compared to single trees
  • Handles both numerical and categorical features
  • Provides feature importance rankings
  • Works well for classification and regression tasks
  • Default: 100 estimators, max depth of 10

Gradient Boosting Classifier

Type: Sequential Ensemble Learning

Best For: Highest accuracy, complex decision boundaries

  • Builds models sequentially, each correcting previous errors
  • Excellent for binary and multi-class classification
  • Automatically handles feature interactions
  • Robust to outliers and missing data
  • Provides probability estimates for predictions
  • Default: 100 estimators, max depth of 5

K-Means Clustering

Type: Unsupervised Learning Algorithm

Best For: Customer segmentation, data grouping

  • Identifies natural groupings in your data
  • Finds k distinct clusters based on similarity
  • Discover hidden patterns and segments
  • Useful for market research and segmentation
  • Visualizable on 2D/3D feature spaces
  • Automatically determines optimal number of clusters

Linear Regression

Type: Supervised Learning Algorithm

Best For: Continuous value prediction, trend analysis

  • Models linear relationships between variables
  • Simple, interpretable, and fast
  • Provides coefficient interpretations
  • Excellent for forecasting and trend analysis
  • Evaluated with R¬≤, RMSE, MAE metrics

Isolation Forest (Anomaly Detection)

Type: Unsupervised Anomaly Detection

Best For: Finding outliers and unusual patterns

  • Detects anomalies without labeled data
  • Identifies fraud, errors, or unusual events
  • Works well with high-dimensional data
  • Produces anomaly scores for each record
  • Automatically flags suspicious patterns

üìä Performance Metrics

Classification Metrics

  • Accuracy: Overall correctness percentage
  • Precision: Positive prediction accuracy
  • Recall (Sensitivity): True positive detection rate
  • F1-Score: Harmonic mean of precision and recall
  • Confusion Matrix: Detailed prediction breakdown

Regression Metrics

  • R¬≤ Score: Model fit quality (0-1, higher is better)
  • RMSE: Root mean squared error
  • MAE: Mean absolute error
  • Prediction Range: Min and max predicted values

Clustering Metrics

  • Cluster Sizes: Number of points in each cluster
  • Cluster Characteristics: Mean values per cluster
  • Silhouette Score: Cluster quality measure
  • Visualization: 2D/3D cluster plots
